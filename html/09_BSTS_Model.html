Los modelos Bayesian Structural Time Series (BSTS) representan un enfoque avanzado y flexible para el anlisis de series temporales que combina elementos del modelado estructural clsico con inferencia bayesiana. Esta metodologa, desarrollada inicialmente por investigadores de Google, permite descomponer una serie temporal en sus componentes fundamentales (tendencia, estacionalidad y ciclos), mientras incorpora la incertidumbre en la estimacin de parmetros a travs de distribuciones de probabilidad. Los BSTS se destacan por su capacidad para manejar datos faltantes, incorporar variables explicativas, y realizar predicciones robustas con intervalos de credibilidad, lo que los hace particularmente valiosos en aplicaciones empresariales como la previsin de ventas, el anlisis de marketing y la evaluacin del impacto de intervenciones comerciales.

Modelo BSTS

Los modelos de prediccin de series temporales BSTS (Bayesian Structural Time Series) son una metodologa estadstica que combina tcnicas de series temporales clsicas con un enfoque bayesiano para modelar y predecir datos secuenciales. Estos modelos se basan en la descomposicin de las series temporales en componentes estructurales, como tendencia, estacionalidad y ciclos, permitiendo as una interpretacin ms clara y flexible. Al ser bayesianos, los modelos BSTS proporcionan una estimacin probabilstica que permite incorporar la incertidumbre de las predicciones, lo que los hace tiles en situaciones donde los datos son ruidosos o cuando se requiere un enfoque robusto para capturar patrones complejos.

def train_bsts_and_predict(df, test_size=15, sp=12):
    # Split the data into train and test sets
    train, test = df[:-test_size], df[-test_size:]
    
    # Define the model
    model = BSTS(seasonality=sp)
    
    # Fit the model to the training data
    model.fit(train,
            num_warmup = 200, # 2000
            num_samples = 200, # 2000
            num_chains= 2)
    
    # Generate predictions
    predictions = model.predict(len(test))
    
    # Ensure predictions have the same shape as test
    predictions = predictions[:len(test)].flatten()
    
    # Plot the results
    plt.figure(figsize=(12, 6))
    plt.plot(train.index, train, label='Train')
    plt.plot(test.index, test, label='Test')
    plt.plot(test.index, predictions[:len(test)], label='Prediction')
    plt.legend()
    plt.show()

Conclusin BSTS

Las predicciones generadas por los modelos BSTS (Bayesian Structural Time Series) no han alcanzado el nivel de precisin esperado y el coste computacional ha sido considerablemente alto debido al tiempo requerido para entrenarlos. Esto puede deberse a la complejidad de los datos y la especificacin del modelo, que no capturan adecuadamente todas las dinmicas subyacentes de la serie temporal. Al comparar con otros enfoques ms simples como ETS o Theta, hemos encontrado que estos pueden ofrecer predicciones mejores con un coste computacional mucho menor. Por lo tanto, se recomienda considerar modelos alternativos que ofrezcan un mejor equilibrio entre precisin y eficiencia computacional, realizando un anlisis exhaustivo de los datos y experimentando con diferentes configuraciones de modelos.

El coste computacional ha sido considerablemente alto debido al tiempo requerido para entrenarlos. Este tradeoff entre la alta capacidad de cmputo y la precisin de las predicciones es una desventaja significativa, ya que la complejidad del modelo y la estimacin bayesiana de los parmetros no siempre resultan en mejoras sustanciales en la exactitud de las predicciones.

BSTS¶

Autor: Ramón Morillo Barrera

Dataset: Madrid.csv¶

Los modelos Bayesian Structural Time Series (BSTS) representan un enfoque avanzado y flexible para el análisis de series temporales que combina elementos del modelado estructural clásico con inferencia bayesiana. Esta metodología, desarrollada inicialmente por investigadores de Google, permite descomponer una serie temporal en sus componentes fundamentales (tendencia, estacionalidad y ciclos), mientras incorpora la incertidumbre en la estimación de parámetros a través de distribuciones de probabilidad. Los BSTS se destacan por su capacidad para manejar datos faltantes, incorporar variables explicativas, y realizar predicciones robustas con intervalos de credibilidad, lo que los hace particularmente valiosos en aplicaciones empresariales como la previsión de ventas, el análisis de marketing y la evaluación del impacto de intervenciones comerciales.

Modelo BSTS¶

Los modelos de predicción de series temporales BSTS (Bayesian Structural Time Series) son una metodología estadística que combina técnicas de series temporales clásicas con un enfoque bayesiano para modelar y predecir datos secuenciales. Estos modelos se basan en la descomposición de las series temporales en componentes estructurales, como tendencia, estacionalidad y ciclos, permitiendo así una interpretación más clara y flexible. Al ser bayesianos, los modelos BSTS proporcionan una estimación probabilística que permite incorporar la incertidumbre de las predicciones, lo que los hace útiles en situaciones donde los datos son ruidosos o cuando se requiere un enfoque robusto para capturar patrones complejos.

def train_bsts_and_predict(df, test_size=15, sp=12):
    # Split the data into train and test sets
    train, test = df[:-test_size], df[-test_size:]
    
    # Define the model
    model = BSTS(seasonality=sp)
    
    # Fit the model to the training data
    model.fit(train,
            num_warmup = 200, # 2000
            num_samples = 200, # 2000
            num_chains= 2)
    
    # Generate predictions
    predictions = model.predict(len(test))
    
    # Ensure predictions have the same shape as test
    predictions = predictions[:len(test)].flatten()
    
    # Plot the results
    plt.figure(figsize=(12, 6))
    plt.plot(train.index, train, label='Train')
    plt.plot(test.index, test, label='Test')
    plt.plot(test.index, predictions[:len(test)], label='Prediction')
    plt.legend()
    plt.show()

Conclusión BSTS¶

Las predicciones generadas por los modelos BSTS (Bayesian Structural Time Series) no han alcanzado el nivel de precisión esperado y el coste computacional ha sido considerablemente alto debido al tiempo requerido para entrenarlos. Esto puede deberse a la complejidad de los datos y la especificación del modelo, que no capturan adecuadamente todas las dinámicas subyacentes de la serie temporal. Al comparar con otros enfoques más simples como ETS o Theta, hemos encontrado que estos pueden ofrecer predicciones mejores con un coste computacional mucho menor. Por lo tanto, se recomienda considerar modelos alternativos que ofrezcan un mejor equilibrio entre precisión y eficiencia computacional, realizando un análisis exhaustivo de los datos y experimentando con diferentes configuraciones de modelos.

El coste computacional ha sido considerablemente alto debido al tiempo requerido para entrenarlos. Este tradeoff entre la alta capacidad de cómputo y la precisión de las predicciones es una desventaja significativa, ya que la complejidad del modelo y la estimación bayesiana de los parámetros no siempre resultan en mejoras sustanciales en la exactitud de las predicciones.
